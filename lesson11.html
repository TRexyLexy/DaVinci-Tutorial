Lesson 11: RF Emulator

Step 1 of 10

Downloading the RF Emulator

In this lesson, we will learn to download and set up a vuirtual device which mimics a real RF device. This is useful for learning and testing however, it is not required.

If you prefer to a real RF device or simply do not need the emulator, you may skip this lesson.

DaVinci APK

    Download the DaVinci App Files using the link below:

  • DaVinci Downloads

    • Click on the most recent link at the top to download the APK File


Android Studio

Next, download Android studio using the link below:

  • Android Studio

    • Click on the white download button in the center of the screen to begin the download.


    Install Android Studio

    To install Android Studio, simply click "next" and "finish" at the bottom of the screen. There is not need to change any setting while in the instalation screen.


    After completing the installation process, open the Android Studio application.
    You should then see the projection creation screen.


    Emulator

    In the project creating screen, select "No Activity" and then next and finish to create the project.


    Once in the project screen click on the menu in the left corner then select "Tools" and "Device Manager."

    Setup

    In the device manager tab, click the plus button to add a new device.


    Next, click "Create Virtual Device" and then choose the "Pixel 9 Pro" as your device.


    Dispatch Management

    There is no need to change any settings within the device configuration, simply select "Finsih."


    Start the device by clicking the play button beside your device.


    Allow a few moments for the device to start up.
