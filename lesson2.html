Lesson 2: Setup and Importing Data

Warehouse Setup

Setting up a warehouse in DaVinci WMS starts by navigating to the "Warehouse Terminal" and clicking the green Insert button. When creating a new warehouse, the only mandatory fields are:

  • Description
  • Mode
  • Bounded/Unbounded

You can also fill in additional fields to provide more detailed information. Remember to click Save after entering the data. You can return at any time to edit the warehouse details.
