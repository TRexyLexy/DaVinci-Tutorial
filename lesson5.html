Lesson 4: Billing

Step 1 of 5

Viewing Accounts for Billing

To view accounts for billing, navigate to the dashboard and the dropdown menu.

Select Business Partners, then Accounts.

Charge Codes

Select an account and click Charge Codes on the left sidebar.

To create a new charge code, press the green insert button.

The charge code menu offers many different type options to choose from.

Assigning Charges (3-Step Process)

  1. Open an Order.
  2. Enter Manual Charge Codes using the input fields for values (default is zero).
  3. Verify Automatic Charge Codes, which are system-applied and appear on the right side.

Storage Charges & Default Billing

There are numerous type options for storage charges depending on business rules.

Default billing provides a simple way to set up billing for accessorial and storage for all units of measure.

Monthly Billing & General Billing Settings

Monthly billing allows specific recurring fees to be applied each month in addition to storage fees.

The Billing section also includes invoice configuration and modes for renewal storage billing.
